## 📌 Sobre o Projeto

Este projeto foi desenvolvido como **trabalho de extensão** com propósito duplo: aplicar na prática os conteúdos de Design Digital vistos em sala e, ao mesmo tempo, entregar valor real para um cliente — um salão de cabeleireiro que poderá usar o site como sua presença digital oficial.

A proposta visual é uma landing page no estilo **luxury dark editorial**: fundo escuro, tipografia serifada, detalhes em dourado e animações sutis que transmitem sofisticação sem perder usabilidade.

---

## ✨ Funcionalidades

- **Navbar** fixa com efeito de blur e transição ao rolar a página
- **Hero Section** com headline impactante, estatísticas e call-to-action
- **Seção de Serviços** com cards interativos e hover animado
- **Galeria** em grid assimétrico estilo editorial
- **Seção Sobre** com layout de duas colunas
- **Depoimentos** de clientes com cards estilizados
- **CTA de Agendamento** com link direto para WhatsApp
- **Footer** completo com horários, links e redes sociais
- **Scroll reveal** nas seções ao rolar a página
- **Responsivo** para desktop, tablet e mobile

## 🎨 Decisões de Design

A identidade visual foi pensada para transmitir **sofisticação e confiança**, atributos importantes para um salão de alto padrão.

**Paleta de cores:**

| Token | Hex | Uso |
|---|---|---|
| `--color-black` | `#0D1420` | Fundo principal |
| `--color-navy` | `#1A2438` | Fundo secundário |
| `--color-charcoal` | `#2C3547` | Superfícies e cards |
| `--color-gray` | `#8A909E` | Textos secundários |
| `--color-paragraf` | `#4d5158` | Parágrafos |
| `--color-gold` | `#C9A84C` | Destaques e CTAs |
| `--color-offwhite` | `#F5F2ED` | Textos principais |

**Tipografia:**
- **Playfair Display** — títulos e destaques (elegância e sofisticação)
- **DM Sans** — corpo de texto (leitura limpa e moderna)
